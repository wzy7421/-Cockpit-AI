# Data Folder

This folder contains manuscript-level summary CSV files and small synthetic sample CSV files. The full synthetic example dataset is generated by running:

```bash
python scripts/generate_synthetic_data.py
```

After running the script, the following full CSV files are created locally:

- `participants_synthetic.csv`: 192 synthetic participant-level records.
- `trials_synthetic.csv`: 2,304 synthetic trial-level records.

The included sample files are:

- `participants_synthetic_sample.csv`: first 16 synthetic participant rows for schema inspection.
- `trials_synthetic_sample.csv`: first 64 synthetic trial rows for schema inspection.

The summary files correspond to manuscript-level aggregate results and can be inspected directly without generating the full synthetic files.
